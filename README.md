# ansi-playbook-webserver-centos7
Installing Webserver on Centos 7 using Ansible AWX Tower

This Playbook works:
- Disabled SELinux
- Stop and disable firewalld
- Install Repository epel-release yum-utils remi-release
- Enable repo and install php7.4
- HTTPD
- MariaDB-Server
- memcached
- wget
- links
- cifs-utils
- nano
